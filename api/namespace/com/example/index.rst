com.example
===========

The ``com.example`` namespace serves to hold XBR API examples and cannot be registered by users. It is maintained as part of this repository.

.. note::

  The `example.com <https://example.com/>`_ domains is officially reserved
  and registered by the IANA to serve as a safe and collision free way
  for Internet related examples that need a domain - like here.
